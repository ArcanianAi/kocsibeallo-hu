# Reports Directory

This directory contains timestamped reports generated by diagnostic and fix scripts.

## Generated Reports

### Diagnostic Reports
- **Pattern:** `production-diagnostic-YYYYMMDD_HHMMSS.txt`
- **Generated by:** `./scripts/diagnose-production.sh`
- **Contains:** Full diagnostic of production server including git status, file sizes, permissions, config state

### Fix Reports
- **Pattern:** `fix-deleted-files-YYYYMMDD_HHMMSS.txt`
- **Generated by:** `./scripts/fix-deleted-files.sh`
- **Contains:** Log of file restoration process and verification results

## Report Retention

Reports are excluded from Git (see `.gitignore`) and should be kept locally for debugging purposes.

Old reports can be safely deleted when no longer needed.
