## üéØ Project Overview

**Project:** Kocsibeallo.hu - Drupal 7 ‚Üí Drupal 10 Migration
**Environment:** Docker-based local development

This project involves migrating the Kocsibeallo.hu website from Drupal 7 to Drupal 10, including:
- Content migration (nodes, taxonomy, users, webforms)
- File and media migration
- Porto theme adaptation and customization
- Module upgrades and replacements
- Site configuration and optimization

**Quick Access URLs:**
- **Live Site:** https://www.kocsibeallo.hu
- **D10 Dev:** http://localhost:8090 (admin / admin123)
- **D7 Dev:** http://localhost:7080 (reference)
